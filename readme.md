Emergency Call App allows users to quickly call emergency numbers, track call usage with credits, view call history, and like important contacts.

[Live Demo](https://sakibulla.github.io/JS-AS1/)

---

## Features

- Call emergency numbers directly from the platform.
- Users have a credit system, which decreases with each call.
- Call history to track past calls.
- Users can like important contacts for quick access.
- Simple and easy-to-use interface for emergencies.

---

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- No backend required (static site)

---

## Dependencies

- Pure JavaScript (no external libraries)
- Optional: CSS frameworks if used (e.g., Bootstrap)
